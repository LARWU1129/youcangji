# 邮藏集 - PWA 邮票收藏管理 App

基于 Figma 设计稿构建的渐进式 Web 应用。

## 本地运行



然后用手机浏览器打开 http://<电脑IP>:8080，点击安装到主屏幕。

## 项目结构

├── index.html          # 主页面
├── manifest.json       # PWA 配置
├── service-worker.js   # 离线缓存
├── css/style.css       # 样式
├── js/app.js           # 应用逻辑
└── icons/              # 图标

## 功能

- 首页：统计、最近浏览、快捷操作、拍卖动态
- 市场：邮票浏览、搜索、分类筛选
- 收藏：我的收藏列表
- 详情：藏品信息、拍卖纪录、收藏切换
- 录入：新增藏品表单
- 我的：个人中心
